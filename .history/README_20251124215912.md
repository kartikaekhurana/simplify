# Resume Keyword Extractor

Extract keywords from resume PDFs using Python.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Command Line

```bash
python keyword_extractor.py path/to/resume.pdf
```

### As a Module

```python
from keyword_extractor import extract_keywords_from_resume

result = extract_keywords_from_resume("resume.pdf")
print(result)
```

## Features

- Extracts text from PDF files
- Identifies technology keywords (Python, React, AWS, etc.)
- Extracts general keywords using NLP (nouns, proper nouns)
- Returns keyword frequency counts

## Output

Returns a dictionary with:
- `tech_keywords`: List of technology-related keywords found
- `general_keywords`: Dictionary of general keywords with frequency counts
- `total_keywords`: Total number of unique keywords found
